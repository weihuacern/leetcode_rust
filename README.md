# LeetCode Rust

A leetcode practice repo in rust.

## Rust

### Installation

#### MacOS

```bash
brew install rustup
rustup-init
```

### Run

Lint:

```bash
cargo clippy
```

To test all solutions:

```bash
cargo test
```

To test one single solution:

```bash
cargo test test_{question ID}
```

For example,

```bash
cargo test test_0001
cargo test test_0002
```
